60  /*
61  * Initialize only those member variables defined in this class.

63  * values of the state variables in mpOdeSystem as an initial condition.
64  *
65  * The member variables mDivideTime and mG2PhaseStartTime are

68  *

70  * mMinimumGapDuration, mStemCellG1Duration, mTransitCellG1Duration,
