Detailed Description

template<unsigned ELEMENT_DIM, unsigned SPACE_DIM>
class CellLocationIndexWriter< ELEMENT_DIM, SPACE_DIM >

A class written using the visitor pattern for writing cell location indices to file.

The output file is called results.vizlocationindices by default. If VTK is switched on, then the writer also specifies the VTK output for each cell, which is stored in the VTK cell data "Location indices" by default.
