Detailed Description

template<unsigned ELEMENT_DIM, unsigned SPACE_DIM>
class TrapezoidEdgeVertexMeshWriter< ELEMENT_DIM, SPACE_DIM >

A mesh writer class for vertex-based meshes with edge representation.
